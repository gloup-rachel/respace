System 06

UTM 생성기

URL 입력 → 캠페인 자동 추론 → 매체 선택 → UTM URL 즉시 생성.
뉴스레터·블로그·카드뉴스·광고 채널 전부 지원.

📖 처음 쓰시나요? 3단계 사용 설명서
펼쳐보기 ▾ 접기 ▴
Step 01
🔗

URL 붙여넣기

링크 걸 페이지 주소를 입력하면, 캠페인이 자동으로 감지됩니다.
예: 팝업 페이지 → 05_popup 자동 추론

Step 02
📣

채널 선택

콘텐츠를 올리는 곳을 클릭하세요.
뉴스레터 · 블로그 · 카드뉴스(인스타) · 카카오 · 네이버 광고 등 source/medium이 자동으로 채워집니다.

Step 03
✅

생성 & 복사

UTM 생성 버튼을 누르면 완성된 URL이 바로 나옵니다.
복사 버튼으로 바로 쓰거나, CSV로 내려받으세요.

콘텐츠 팀 자주 쓰는 채널 — 이렇게 채우면 됩니다

채널매체 선택캠페인 코드 예시utm_term (식별자)
📧 뉴스레터 뉴스레터 newsletter_202605_01 (연월_호수) 01, 02 (호수)
📝 블로그 블로그 blog_202605_popup-trend (연월_슬러그) 글 슬러그 (예: popup-trend-2026)
📸 카드뉴스 (인스타) 인스타 오가닉 instagram_202605_01 (연월_번호) 게시물 번호 또는 슬러그
💬 카카오 채널 카카오 kakao_20260526 (발송일) 발송일 YYYYMMDD
🔗 링크드인 링크드인 linkedin_202605_01 게시물 번호

예시 — 뉴스레터 5월 1호 UTM

입력 URL https://respace.co.kr/INTRO-WORKS
매체 선택 뉴스레터
캠페인 newsletter_202605_01
결과 URL https://respace.co.kr/INTRO-WORKS?utm_source=newsletter&utm_medium=email&utm_campaign=newsletter_202605_01&utm_content=cta_button
