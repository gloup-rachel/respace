## 인사이트 생성기 실행 방법

### Step 0. 준비 — 두 파일 로드

Claude에게 인사이트를 요청하기 전에 아래 두 파일을 함께 첨부하거나 내용을 공유한다.

    1. 퍼포먼스_인사이트_분석_기초.md
    2. [클라이언트명]_인사이트_설정.md  (예: 리스페이스_인사이트_설정.md)

### Step 1. 데이터 제공

아래 형식 중 하나로 데이터를 제공한다.

    (A) 대시보드 스크린샷 + 핵심 수치 텍스트 요약
    (B) 엑셀/CSV 원본 파일 첨부
    (C) 수동 입력 (아래 데이터 입력 템플릿 사용)

### Step 2. 생성 요청 프롬프트

Claude에게 아래 프롬프트를 그대로 사용한다.

- [클라이언트명]_인사이트_설정.md: 클라이언트 KPI, 채널 구조, 벤치마크

[분석 데이터]
(여기에 데이터를 붙여넣거나 파일 첨부)

[요청 사항]
아래 구조로 리포트를 작성해줘:

1. 이번 주 한 줄 요약 (KPI 기준)

6. 다음 주 체크 포인트

분석 원칙:
- 모든 성과는 PRIMARY KPI 기준으로 먼저 판단
- 팩트는 수치로 명시, 단정 금지
- 원인은 [확인됨] / [추정] 구분
- 액션은 임팩트 큰 것부터

## 출력 예시 (리스페이스 W20 샘플)

    [W20 한 줄 요약]
    전체 CPL 32,400원으로 전주 대비 12% 상승. Google SA 집행 효율 저하가 주원인.

    [전체 성과]
    - CPL: 32,400원 (전주 28,900원, +12.1%)
    - 리드: 23건 (전주 26건, -11.5%)
    - 총 비용: 745,000원 (전주 751,000원, -0.8%)

    [매체별]
    - Google SA: CPL 38,200원 (+22%) — 이상 신호 감지, 딥다이브 진행
    - Naver SA: CPL 24,100원 (-3%) — 정상 범위
    - Meta: CPL 29,800원 (+5%) — 정상 범위

    [딥다이브 — Google SA]
    LEVEL 2 확인: '서비스 검색' 캠페인 CPL 41,500원 (+34%)
    LEVEL 3 확인: '팝업스토어 대행' 광고그룹 CPC 2,800원 → 4,100원 (+46%)
    → [확인됨] CPC 급등이 CPL 상승의 직접 원인 (전환율은 동일 수준 유지)
    → [추정] 경쟁 입찰가 상승으로 추정 (Auction Insights 미확인)

    [액션 아이템]
    [즉시]     Google SA '팝업스토어 대행' 그룹 입찰가 상한 3,200원으로 하향
    [검증 후]  Auction Insights 확인 후 경쟁사 동향 파악
    [중장기]   CVR 유지하면서 CPC 낮은 유사 키워드 발굴 및 테스트

    [다음 주 체크]
    - Google SA '서비스 검색' 캠페인 CPL 전주 대비 회복 여부
    - Naver SA 유지 추세 지속 확인
    - Meta 5% 상승이 추세인지 노이즈인지 2주 추이 관찰

## 다른 클라이언트 확장 방법

리스페이스 외 파트너사에 적용할 때:
1. [클라이언트명]_인사이트_설정.md 파일을 새로 만든다.
2. 설정 파일에 해당 클라이언트의 KPI, 채널, 캠페인 구조, 벤치마크를 채운다.
3. 인사이트 생성 요청 시 해당 설정 파일로 교체해서 사용한다.
